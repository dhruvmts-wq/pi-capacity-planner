PI Capacity Calculator

SAFe Scrum · 8 members · 5 iterations + IP
Exported!
This calculator computes available capacity (story points & hours) for each iteration in your PI, factoring in individual focus, meeting load, PTO, and org holidays. All data stays in your browser — nothing is sent anywhere.

Quick start

Work through the tabs left to right: Team Setup → Iterations → Holidays & PTO → Summary. The Summary tab auto-refreshes whenever you switch to it.

Team Setup tab

Eight members are pre-loaded (6 Devs, 2 DITs). Click any member card to expand and configure focus factor, meeting overhead, and experience level.

Iterations tab

Set start/end dates for each iteration. Working days (Mon–Fri, excluding org holidays) are calculated automatically.

Summary tab — Work Type Allocation

The Summary page hosts a fully editable Work Type Allocation panel. You can add any custom work type (Feature, RTR, TCoO, etc.), set overall % split, and override individual iteration percentages. A validation check ensures each iteration's work types sum to 100% — an error banner appears if not. Story points per work type are computed automatically from each iteration's total SP.

Holidays & PTO tab

Add organisation-wide holidays (applied to all members) and individual PTO days per iteration.

Export PDF

Click Save as PDF Report on the Summary tab to generate and download a formatted PDF report containing PI totals, work type split, per-iteration breakdown, and per-member capacity table.

How story points are calculated

1 story point = 8 hours (fixed). SP and hours are always directly proportional.

Available days = working days − org holidays − PTO days
Effective hours = available days × 8h × (1 − meeting overhead%)
Story points = effective hours ÷ 8
Pro tip: Run at the start of PI Planning with best-guess PTO. Update it as PTO is confirmed during the PI and re-check the Summary to guard against over-commitment.
